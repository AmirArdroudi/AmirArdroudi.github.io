    About Me

    I'm AmirAhmad Ardroudi, a game programmer/developer from Iran.

    From my childhood I loved playing video games, and that derived a passion towards making them. Then, I set about building games when I was a teenager using Unity and Unreal game engines. In college I decided to pursue my career as a game developer, because of that, since then I have built (or worked on) eight projects.

    Beside developing games, I have some experience in game engine programming and architecture, such as Entity Component System (ECS) and Data Driven design.

    Furthermore, throughout these years, I have kept a mindset of programming clean, consice, resource-friendly and optimized code. Also I have a good deal of experience working with teams and collaborating with game designers which had a palatable impact on player experience and leads to juicing up the aformentioned projects.

    Ariyana Game Engine

    Ariyana is a data-driven modular game engine which uses fiber system for enhancing multi-threading and provides scripting possibility with beef language. In this project I was responsible for creating 2D system module and its scripting wrapper around beef.

    • C++
    • BeefLanguage
    • Data-Driven
    • Android/PC

    Block Heroes

    Block Heroes is an experimental game based on Tetris, built with Ariyana engine, which has online multiplayer capability and replay system (you can watch back the entire game that you played). It can be played Block Heroes in GameJolt.

    • C++
    • Ariyana Engine
    • ECS
    • Multiplayer
    • Android/PC

    Unity Custom Event Handler

    Programmed a modular system for handling events based on scriptable-object and UnityEvent in unity engine. Developer/Designer could hook things up in editor, which comes handy in gameplay development and will help to isolate prefabs. The idea of this project was inspired by some conferences on game development which is referenced on the github repo.

    • Unity
    • C#
    • Scriptable-Objects
    • modular

    Ari2D-Tutorial

    Created the 2D system for Ariyana engine and written setup documentation for 2D module system around beef language IDE in form of a tutorial. Which explains setting up the environment and scripting a 2D project.

    • C++
    • BeefLanguage
    • ECS
    • Game-Engine
